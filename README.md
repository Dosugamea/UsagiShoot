# UsagiShoot
魔法少女チノが悪い子を焙煎する横スクロールシューティングゲーム(予定)

## 仕様
- TypeScript/Phaser3使用
- 横スクロールシューティング
- マウス/キーボード/タッチ対応
- 自機は魔法少女チノ(あとで描く)
- 敵はクロモン的なものと、既存の隕石システムを利用
